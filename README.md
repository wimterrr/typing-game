# Key Tempo

한국어, 영어, 혼합 문장으로 플레이할 수 있는 정적 웹 타자게임이다.

## Features

- 30초, 60초, 90초 타이머
- 한글, 영문, 혼합 문장 모드
- 속도, 정확도, 실수, 콤보, 최고 점수 표시
- 반응형 레이아웃
- `localStorage` 기반 최고 점수 저장

## Run

```bash
npm install --include=dev
npm run dev
```

## Build

```bash
npm run build
```

빌드 결과물은 `dist/`에 생성되며 Vercel, Netlify, Cloudflare Pages 같은 정적 호스팅에 바로 배포할 수 있다.
